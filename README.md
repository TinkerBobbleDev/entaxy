# Bean Counter

A personal finance app.

[![Build Status](https://travis-ci.org/bean-counter/bean-counter.svg?branch=master)](https://travis-ci.org/bean-counter/bean-counter)


## Running the server

```
npm install
npm start
```

## Built with

* ReactJS
* Webpack
* Blockstack


## Tests

We're using Jest and you can run the tests with

```
npm test
```
